Login Back To Top

  • Q: How do I change my password?
    • A: Once you're logged in, click on you name on the top-right of the screen, and select the menu "Settings". For more detail, please watch the following:

  • Q: I'm logged in as 'Sandbox User.' Is that normal?
    • A: 'Sandbox User' is a special user used to let people get a quick look at what an annotated PDF looks like in NB. But in order to really use NB, you need to be logged in as yourself. So logout by chooing "Log out" from the top right menu, or visiting the /logout URL (e.g. nb.mit.edu/logout if you're using the nb.mit.edu server for instance), and re-login as yourself.
  • Q: Is there a way to recover my password if I forgot it?
    • A: You can ask for a password reminder near the login box.

Memberships Back To Top

  • Q: How do I get access to contents from a class on NB?
    • A: It depends on that class settings:
    • If the class is "by invitation only", you need to contact to the class administrator (i.e. the instructor or the TA in most cases) and ask them for an invitation.
    • If the class provides a "subscribe link", you just need to enter that "subscribe link" on your browser and confirm that you want to be part of that class on NB.
    • If the class material is viewable by the whole world you don't even need an account on NB: You just need to visit the URL of any of that class (it looks like http://[SERVER_NAME]/f/[SOME_NUMBER], for example http://ocw.csail.mit.edu/f/1)
    For more detail, please watch the following:

  • Q: Can I remove myself from a class?
    • A: Not currently: see https://github.com/nbproject/nbproject/issues/13 If you need to be removed from a class in the meantime, please contact us directly using the email that's registered in NB and telling us the class you need to be removed from.

Troubleshooting Back To Top

  • Q: How can I get help with an NB-related question?
    • A:
    1. First check if your question is on this FAQ.
    2. Not there ? Visit our forum
    3. As a last resort, you can contact us at nb-team@csail.mit.edu (note: you'll likely get a much faster response if you use the forum above since we don't have the infrastructure to have a dedicated support team).
  • Q: Nothing shows up on the main screen and none of the links work?
    • A: Are you a compatible browser ? Currently NB works with the following browsers:
    • Firefox
    • Safari
    • Chrome
    • Internet Explorer 9 or more recent. If you're using on of these browser and that NB doesn't work, please visit our forum.

  • Q: Is my information secure? What gets stored where? What about privacy?
    • A:
    • If you're using the http://nb.mit.edu server, NB stores your annotations and your reading history on a database located on the MIT campus.
    • There are 3 visiblility settings for your notes:
      • The Entire Class, meaning that those notes can be seen by both students and staff.
      • Staff only, meaning those notes can be seen by the staff. Note that some NB sites might not allow Staff Only notes.
      • Myself only notes are private: Staff and other students can't see them

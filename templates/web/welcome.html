Welcome to NB!

An online collaborative annotation tool made for the classroom.

Make a note anywhere.

Click and drag a box to highlight anywhere in the document: text, graph, drawing, anything.

Anchored notes to text.

Make a note right next to the highlighted text, knowing which part of the document the comment refers to.

Harness class wisdom.

Students can see notes posted by their classmates, ask questions and answer each others' questions.

See trouble spots.

Instructors can tell which parts of the documents need more attention by seeing which part got more comments.

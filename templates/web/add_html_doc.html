Add a new webpage to NB

This is an experimental feature... NB can be used to comment simple HTML pages.
  • Use this form to indicate the page that you'd like to comment with NB
  • That page needs to contain our embedded NB viewer. To do so, you need to insert the following snippet into the <head> section of your page:
    <script src='HOSTNAME/embed_NB.js' type='text/javascript'> </script>
  • If you can't add the embedded NB viewer for any reason (for instance, you don't own the page), you can tell your users to drag the NB bookmarklet into their bookmarks toolbar to insert the script. In order to communicate this bookmarklet to your users, you can give them this link to our FAQ.

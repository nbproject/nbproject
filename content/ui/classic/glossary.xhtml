Glossary (work in progress...)

  • Active page
    This is the page for which annotations are currently being displayed, and that you can annotate by dragging a area on the PDF with your mouse.
  • Collage View
    A collage view displays a collection of threads along with some preview of the material that is related to that annotation. Unlike the Document View, the Collage View focuses on the annotations rather than on the material itself. The figure below shows a sample Collage View.
  • Context Menu
    nb uses context menus to avoid UI clutter. When you're on any note of an expanded thread, you can open the context menu by right-clicking on that note (mac users: Control+click) or by clicking on the options... link located in that note.
  • Document View
    The Document View displays a PDF document in the browser (similar to a regular PDF reader) along with annotations.
  • Favorites
    A comment that you've marked as favorite, using the Context Menu. You can choose to see only favorite comments as you read along. From the home page you can monitor new ongoing discussions related to your favorite comments.
  • Full text search
    All your annotations on nb are searchable (but the contents of the PDFs themselves isn't searchable yet). Use the following tips to improve your searches.
    • Avoid substrings that aren't real words in the search bar. nb uses an index optimized for search expressions that are real English words. For example, searching for cla may not return any result, but searching for class may return hundreds of results.
    • The order of the search terms doesn't matter. For instance, searching for middle class and class middle will return the same results.
    • Use your own strings to create your bookmarks, tags and collections. nb doesn't have a feature for users to group their annotations using custom tags or collections (yet...). However, you can simulate that feature in the following way: Assuming you've found an interesting place in the lecture, just write a private annotation on that place (or write a private reply to an exisiting note, if there's already a note at that place) .This new annotations should contain a unique string that you will easily remember made of letters, digits, and the hyphen character "-" (e.g. review-for-6042-final). You can then use this string in the nb search bar to find all the annotations that you've marked with that string.
  • Global Comment
    A type of note that isn't related to any particular location in the text. Writing global comments are appropriate to write reflections on the document as a whole, rather that focusing on some precise locations in the material.
  • Note
    A single annotation written about the open document. There are essentially 3 ways to create a note.
    1. To create a new thread about some region of the PDF material, drag your mouse cursor over that region of the PDF. An in-place editor will appear, where you can type your note, and save it.
    2. To create a global comment on that PDF, click on or or the "New..." link located on the top right corner of the document
    3. To reply to an existing note, click the Reply button on the context menu.
  • Thread
    Threads are conversations, and they consists of one or many notes about the same location in the text. You can add a note to a thread by clicking the Reply button on the context menu. By default, NB shows you many threads in Collapsed Form (indicated by a '>'). To expand a thread, just click anywhere on that thread.
    Note that expanding a thread also highlights and scrolls the PDF so that you can see what location in the material this thread discusses.
